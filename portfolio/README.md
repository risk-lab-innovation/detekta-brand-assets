## Which file do I use?

| Situation | File |
|---|---|
| Website / app on dark background | `SVG/detekta_main_dark.svg` (or `PNG/detekta_main_white_*.png`) |
| Documents, email, light backgrounds | `SVG/detekta_main_light.svg` (or `PNG/detekta_main_ink_*.png`) |
| Profile pictures, avatars, app icons | `Social/avatar_ink_1024.png` (square, ink background baked in) |
| Small spaces ≥ 40px (signatures, badges) | `SVG/detekta_short_*.svg` / `PNG/detekta_short_*.png` |
| Browser favicon / tiny UI < 40px | `favicon.ico` or `PNG/favicon_*_32.png` (simplified cut: one D, one swoosh) |
| iOS/Android home icon | `PNG/favicon_ink_180.png` (apple-touch) or `Social/avatar_ink_512.png` |
| Print / designer handoff | The `SVG/` files — infinitely scalable vectors |

Naming: `white` = white wordmark/D, use ONLY on dark backgrounds · `ink` = dark wordmark/D, use on white/light backgrounds.

## Rules
- Never stretch, recolor, add shadows/glows, or set "Detekta" in a live font — the wordmark is a drawn vector.
- Full logo needs clear space around it (≈ the width of the D's spine on all sides).
- Below 40px, always use the favicon cut — the full three-swoosh logo turns to mush at small sizes.
- The three swoosh/sparkle colors are semantic (Detect/Decide/Defend) — never repaint them.
